PROJECTS

These projects were all created by me with the use of HTML, CSS, JavaScript, Adobe Illustrator, and GitHub. Built with responsive design, mobile first, and cross-browser compatibility.



Galactic Odometer

2014 ~ 2015

A JavaScript powered odometer that calculates and outputs a running count of the space miles you've traveled as the Earth and galaxy orbit. Complete with cool animations and a space-styled explanation.

The Freshest High

June, 2015

Product description website for new start-up, TABE, LLC. A playful animation adds levity to the home page. Original icons and photographs assist a detailed and slide-styled explanation of product info, instructions. and safety information.

The Way Unraveled

March, 2015

My playful code laboratory, with an emphasis on inspiration and compassion. Plenty of CSS and JavaScript DOM manipulation experiments in here.

The Decid-o-matic 5000

2014 ~ 2015

An interesting question and answer machine. Think of something you are considering doing, answer a few simple questions and enjoy a little JavaScript powered advice. Designed entirely with CSS. No images.

Gianni's Gallery

July, 2015 ~ Present

A simple but elegant horizontal display with an auto scroll feature set to Pachebel's Cannon. Art by Gianni Vardanega, 3D Lighting Artist. Doesn't work in mobile or tablet yet.
